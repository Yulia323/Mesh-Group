IQ Balance

IQ Balance — это доступный бесплатный сервис для потери веса без вреда для здоровья и выстраивания правильного питания с помощью искусственного интеллекта и личного диетолога.

IQ Balance предосавляжки ежедневного баланса калорий и физической активностиет онлайн-инструменты для распознавания калорийности продуктов по фото и поддер, а также даёт доступ к интегрированной доставке полезных продуктов из супермаркетов и готовых блюд из ресторанов без наценки посредников.

Приложение предполагает 2 модели работы:

  1. Под брендом IQ Balance — предоставление услуг клиентам в секторе B2C и B2B: объединение диетологов с пользователями на нашей площадке для продажи программ питания и полного сопровождения, а также интеграция с продуктовыми сетями / ресторанами и фитнес-центрами.
  2. White label — коробочное решение для использования под вашим брендом: для компаний (фитнес-центров, продуктовых сетей и ресторанов) — внутреннее приложение для клиентов/посетителей; для диетологов — инструмент для удобной работы с клиентами. Полностью упакованное приложение кастомизируется под ваш бренд: название, дизайн, функциональные особенности.
